Original Website

The 'paradise with a view' website functions as a rental website for privately owned condos in Waikiki Beach, Honolulu, Hawaii.

Paradise with a View offers rentals for four condos in two locations.

This website is not aesthetically pleasing and is confusing to navigate.

The website is not easy to use, luckily the condos are available for rental on VRBO as well.

The condos are in a popular location with access to amenities at two trusted hotel chains. The condos have external VRBO links with positive reviews. The condos are trustworthy rentals, but the website does not seem trustworthy.

The design of this website makes the condos seem like untrustworhy rentals.
